# LedgerOne

A corporate-grade accounting and tax prototype for individuals and freelancers.

## Features
- Double-entry ledger
- Personal & Freelancer personas
- US tax profiles: 1040 and 1040-NR
- Income sourcing (US vs Foreign)
- Schedule C draft generation
- Streamlit UI

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
